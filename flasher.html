ESP32-S3 Arduino firmware Web Serial

Install Firmware

V10_Basel

Current Vidiotbox hardware firmware built from the local vidiotbox-hardware project. This writes the bootloader, partition table, boot app, and firmware image.

vidiotbox_V1-RF2_Basel

Connect the board over USB, choose its serial port, then start the install. This writes the bootloader, partition table, boot app, and firmware image.

Before Flashing

  1. Use Chrome or Edge on a desktop computer.
  2. Plug Vidiotbox in with a data-capable USB cable.
  3. If the installer cannot connect, hold BOOT while tapping RESET, then try again.
  4. Keep the browser tab open until the installer reports completion.

SD Card Assets

This browser flasher updates internal ESP32-S3 flash only. Runtime Faceimator images, GIFs, sounds, and SD-card configuration still live on the microSD card.
